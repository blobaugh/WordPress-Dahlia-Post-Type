Adds a custom post type for storing Dahlia information inside of WordPress

== Description ==

The dahlia is a very popular type of flower for enthusiasts and growers alike. 
This plugin provides a custom post type tailored around the needs of individual 
dahlias. If you want to store all the plants in your garden this plugin will help
you do it.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
